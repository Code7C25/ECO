ECO — No estas solo, tu voz importa.

ECO es una aplicación creada para que los estudiantes del colegio puedan reportar de forma segura, confidencial y opcionalmente anónima cualquier situación relacionada con malestar, acoso escolar o violencia.
El sistema conecta directamente al estudiante con el equipo de psicólogos del colegio, quienes reciben, analizan y acompañan cada caso con profesionalismo y respeto.
Además del sistema de reportes, ECO ofrece herramientas de autocuidado como respiración guiada, diario emocional, afirmaciones y recursos breves pensados para ayudar al alumno a regular emociones.

Problema que aborda ECO:
En la escuela, muchos estudiantes no se sienten seguros de hablar con un adulto, sienten vergüenza o miedo a represalias.
Esto provoca que numerosos casos de acoso o malestar no se reporten a tiempo, generando un impacto emocional y social negativo.

Solución propuesta ECO ofrece:
✔ Un espacio seguro y accesible
✔ Comunicación directa con psicólogos
✔ Herramientas de bienestar
✔ Navegación simple
✔ Manual interactivo (opcional)

Cómo usar el sistema:
1) Ingresar a ECO
2) Crear una cuenta (opcional)
3) Reportar una situación
4) Hacé clic en “Reportar”.
5) Completá el formulario con la información que quieras enviar.
6) Enviar el reporte
7) Usar las herramientas de bienestar
8) Manual interactivo (opcional)

Tecnologías utilizadas:
- HTML / CSS / JavaScript
- Intro.js (para el tour interactivo)
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
- PWA (Progressive Web App)
- Diseño responsive para mobile y desktop
