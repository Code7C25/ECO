Manual de Usuario — ECO
📘 Introducción
ECO es una aplicación creada para que los estudiantes del colegio puedan **reportar situaciones de acoso, malestar o violencia** de forma **segura, confidencial y opcionalmente anónima**.
Además, ofrece herramientas de bienestar emocional como respiración guiada, diario emocional y afirmaciones positivas.


🎯 Objetivo del sistema
Brindar un **canal de comunicación seguro** entre los estudiantes y el equipo de psicólogos del colegio, facilitando el acompañamiento profesional y promoviendo el bienestar emocional.

🔑 Cómo acceder
1. Ingresar desde cualquier navegador al enlace:
   **[https://eco-web-ddfaa.web.app/](https://eco-web-ddfaa.web.app/)**
2. Puede utilizarse desde **celular, computadora o tablet**.
3. El usuario puede elegir entre:
   * **Ingresar de manera anónima**
   * **Registrarse** o iniciar sesión para una experiencia personalizada

📌 Guía paso a paso
1. **Página de inicio**
Podés acceder al menú principal desde el icono de opciones.
Desde aquí, ya podés comenzar a navegar **sin registrarte** si elegís la modalidad anónima.
2. **Crear una cuenta (opcional)**
* Ir a “Registrarse”.
* Completar correo y contraseña.
* Confirmar.
3. **Ingresar de manera anónima**
* Simplemente ingresar al sitio.
* Podés usar las herramientas y enviar un reporte sin ningún dato personal.
* El sistema no guarda información del usuario.
4. **Iniciar sesión**
* Ingresar correo y contraseña.
* Habilita funciones adicionales para usuarios registrados.
5. **Enviar un reporte**
1. Abrir el menú.
2. Seleccionar **“Reportar”**.
3. Completar el formulario con la información que desees enviar.
4. Podés elegir hacerlo **anónimamente**.
5. Presionar **“Enviar”**.
6. El reporte llega directamente al equipo de psicólogos.
6. **Usar herramientas de bienestar**
Incluye:
* 🌬 Respiración guiada
* ✏️ Diario emocional
* 💬 Afirmaciones
* 🌿 Recursos breves para regular emociones
7. **Cerrar sesión**
Disponible en el menú principal (solo para usuarios registrados).

❓ Preguntas frecuentes
• ¿Mi reporte puede ser anónimo?
Sí, podés enviar el formulario sin ningún dato personal.
• ¿Quién recibe mi mensaje?
El equipo de psicólogos del colegio.
• ¿Necesito crear una cuenta?
No. El registro es opcional.
• ¿Puedo usar ECO en el celular?
Sí, funciona en cualquier dispositivo con navegador.


⚠️ Errores comunes y soluciones
• No puedo iniciar sesión
* Revisar correo y contraseña.
* Verificar conexión a internet.
• La página no carga
* Actualizar la ventana.
* Comprobar conexión Wi-Fi/datos.
• No se envía el reporte
* Completar los campos obligatorios.
* Revisar la conexión a internet.
